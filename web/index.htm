GeLab - genetic experiment labeler

Repository: GitHub powered with baselib

Latest release: Beta

About

GeLab is genetic experiment labeler. Its aim is to fully or partially automatize uncreative labeling routines after the experiment is done.

It is likely, that the CV part needs a lot of work, please, contact me, when it is not working the way you expect!

It is highly unoptimized and runs in 1 thread. Please feel free to request features or demand optimization. I am open to extending this software. Please contact me at: info@fidli.eu

Usage

Usage: gelab.exe inputfile [-b num_of_blocks] [-m "text" [order]] [-l list beginning_symbol_index -n total_column_count] [-i list] [-I] [-M] [-f] outputfile

 inputfile
  - original TIFF file from experiment

 -b num_of_blocks
  - valid range [1, 50]
  - how many experiments are in inputfile - default: 1

 -m "text" [order]
  - valid 'order' range [1, 255]
  - find mark on place 'order' and put 'text' next to it, mark wont be added, if this is not set, if -M is set, [order] MUST be omitted as it has no effect

 -l list beginning_symbol_index
  - valid 'beginning_symbol_range' range [1, 50]
  - 'list' are ASCII characters (one letter) labeling experiment columns in cycle starting at 'beginning_symbol_index', requires -n to be set too, columns wont be market if this is not set

 -n total_column_count
  - valid range [1, 255]
  - sets the total column count that the experiment has

 -i list
  - which blocks in the experiment to output, indexes separated by comma - default: outputs all
  - valid 'index' range [1, 50]

 -I
  - do not invert colors

 -M
  - crop and outline experiment manually. This option will pop up a window that requires interaction

 -f
  - do not prompt for file overwrite. Overwrite it.

 outputfile
  - where to save the ouput to. If file exists, you will be asked whether to overwrite the file.
